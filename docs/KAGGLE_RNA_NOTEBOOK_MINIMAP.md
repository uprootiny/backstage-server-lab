# Kaggle RNA Notebook Minimap

Auto-generated by:

```bash
make kaggle-minimap
```

Outputs:

- `artifacts/kaggle_rna_notebooks_minimap.json`
- `docs/KAGGLE_RNA_NOTEBOOK_MINIMAP.md`

What it provides:

- mass summary of RNA notebook techniques
- algorithmic minimap (clustered tags + examples)
- concept/shtick buckets
- arrangements of known tricks for recombination

## Recombination Workflow

1. Build minimap:

```bash
make kaggle-minimap
```

2. Inspect technique library:

```bash
make technique-list
```

3. Compose cross-notebook strategy:

```bash
make technique-compose IDS=tbm_ensemble,pairwise_distogram_head,recycling_refinement,confidence_calibration,family_dropout_validation
```

This emits:

- `artifacts/technique_compositions/latest.yaml`
- `artifacts/technique_compositions/composed_experiment.yaml`

4. Execute composed experiment:

```bash
source .venv/bin/activate
labops run artifacts/technique_compositions/composed_experiment.yaml --workers 3
```
